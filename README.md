### Unduh Build (Rekomendasi)
Cara termudah untuk bermain adalah dengan mengunduh build yang sudah jadi dari halaman **[Releases]([LINK_KE_HALAMAN_RELEASES_ANDA])**. Cukup ekstrak dan jalankan!

### Jalankan dari Source Code
Jika Anda ingin melihat kodenya:
1.  Pastikan **Godot Engine v4.x** sudah terinstal.
2.  Clone repositori ini:
    ```bash
    git clone [https://github.com/](https://github.com/)[USERNAME_GITHUB_ANDA]/[NAMA_REPO_ANDA].git
    ```
3.  Buka Godot, pilih `Import Project`, dan cari file `project.godot`.
4.  Tekan **F5** untuk menjalankannya.
